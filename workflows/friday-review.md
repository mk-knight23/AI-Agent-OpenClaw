# Friday Review — OpenClaw

Weekly summary covering code output, content performance, and job search progress.

## Trigger
- **Schedule**: Every Friday at 4:00 PM
- **Manual**: `@OpenClaw run friday-review`

## What OpenClaw Reviews

### Code Output
- Repos created or significantly updated this week
- PRs merged or submitted
- Issues closed

### Content Performance (if content-creator skill installed)
- Posts published this week
- Engagement stats: views, likes, shares
- Best-performing post

### Job Search (if job-hunter skill installed)
- Applications submitted this week
- Responses received
- Interviews scheduled

### Portfolio Health (if portfolio-manager skill installed)
- Total GitHub stars this week
- Portfolio site visits
- New followers

## Example Output

```
📅 Friday Review — Week of Mar 3–7

Code:
  • 2 repos updated  |  1 PR merged to external project
  • Opened 3 issues (external contributions)

Content:
  • 2 posts published (LinkedIn + dev.to)
  • Best performer: "How I automated my job search" — 4,200 views

Job search:
  • 5 applications sent
  • 2 responses received (Anthropic, Linear)
  • 1 interview scheduled (Linear — next Tuesday)

Portfolio:
  • +89 GitHub stars this week
  • Portfolio site: 312 visits
```
