# Monday Briefing — OpenClaw

Start the week with a clear plan. OpenClaw reviews weekend activity and prioritizes your week.

## Trigger
- **Schedule**: Every Monday at 8:00 AM
- **Manual**: `@OpenClaw run monday-briefing`

## What OpenClaw Covers

### Weekend Activity
- New GitHub issues on your repos
- Messages from potential employers or collaborators
- Content performance over the weekend

### This Week's Priorities
- Job applications to follow up on
- Open PRs or issues needing attention
- Content to publish (if draft queue has items)

### Upcoming Deadlines
- Interview scheduled this week
- Application deadlines for target companies
- Conference/event submission deadlines (from your notes)

## Example Output

```
☕ Monday Briefing — March 10

Weekend:
  • +56 stars on your open-source project (viral on HN!)
  • 3 new GitHub issues from new users
  • Email from Vercel recruiter

This week:
  🔴 Reply to Vercel recruiter (received Friday)
  🟡 Interview prep for Linear (Wednesday)
  🟢 Publish post: "Building a CLI agent in TypeScript"
  🟢 Modernize your mcp-client repo (job-hunter flagged it's dated)
```
