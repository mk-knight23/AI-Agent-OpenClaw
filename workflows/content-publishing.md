# Content Publishing Pipeline

Write → Generate hero image → Deploy → Share.

## Trigger
Manual: `@OpenClaw publish blog "topic"`

## Steps
1. Research topic (web search + my repos)
2. Write draft with `content-creator` skill
3. Generate hero image prompt → DALL-E 3
4. Push to mk-knight23.github.io repo
5. Deploy via GitHub Pages
6. Post on LinkedIn + X summary thread
7. Submit to dev.to

## Output
- Blog post live at mk-knight23.github.io/posts/topic
- Social media posts drafted for review
